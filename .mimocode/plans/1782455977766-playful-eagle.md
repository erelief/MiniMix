# MiniMix UI 重新设计计划

## 概述
对 MiniMix 进行 UI 重新设计，统一 Lucide 图标使用，提升视觉品质。创建 `feat/ui-redesign` 分支进行所有修改。

## 修改文件清单
- `public/fonts/` — 新增 Geist 字体 woff2 文件（4 个字重）
- `index.html` — 替换所有内联 SVG 为 `<i data-lucide="...">` 元素
- `style.css` — @font-face 声明、颜色优化、新增设计 token、视觉增强
- `main.js` — 添加 `createIcons()` 调用以渲染 HTML 中的 Lucide 图标

## 步骤

### 1. 创建 Git 分支
```bash
git checkout -b feat/ui-redesign
```

### 2. 字体升级 — 本地自托管 Geist (`style.css`, `public/fonts/`)
- 下载 Geist 字体 woff2 文件到 `public/fonts/`（4 个字重：Regular 400 / Medium 500 / SemiBold 600 / Bold 700）
  - 来源：https://github.com/vercel/geist-font/releases
  - 文件：`Geist-Regular.woff2`, `Geist-Medium.woff2`, `Geist-SemiBold.woff2`, `Geist-Bold.woff2`
- 在 `style.css` 顶部添加 4 个 `@font-face` 声明（`font-display: swap`）
- `style.css` line 22: 替换 `font-family` 为 `'Geist', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif`
- 为 `.btn` 设置 `font-weight: 500`
- 为 `.modal-header span`, `.info-app-name` 设置 `font-weight: 600`
- `.info-app-name` 添加 `letter-spacing: -0.02em`

### 3. 颜色优化 (`style.css` lines 1-13)
- `--accent`: `#e94560` → `#d94060`（降低饱和度）
- `--accent-hover`: `#ff6b81` → `#e8566e`
- 新增 `--error: #e8566e` 替代硬编码的 `#ff6b6b`（约 line 947）
- 新增阴影 token（色调匹配深蓝背景）：
  - `--shadow-sm: 0 1px 3px rgba(10, 10, 30, 0.3)`
  - `--shadow-md: 0 4px 12px rgba(10, 10, 30, 0.4)`
  - `--shadow-lg: 0 8px 32px rgba(10, 10, 30, 0.5)`
- 新增间距/radius token（选择性替换现有硬编码值）

- L109: `&times;` → `<i data-lucide="x">`

在 `style.css` 中添加图标尺寸工具类：
```css
.icon-14 { width: 14px; height: 14px; }
.icon-16 { width: 16px; height: 16px; }
.icon-18 { width: 18px; height: 18px; }
.icon-20 { width: 20px; height: 20px; }
.icon-40 { width: 40px; height: 40px; }
```

### 5. 添加 createIcons() 调用 (`main.js`)

在 `main.js` 顶部添加 Lucide 导入和初始化：
```js
import { createIcons } from 'lucide';
```
在 DOMContentLoaded 或 init 函数中调用 `createIcons()` 渲染 HTML 中的 `<i data-lucide>` 元素。

### 6. 视觉增强 (`style.css`)
- 添加微妙的噪点纹理叠加层（`body::after`，SVG data URI，opacity 0.03）
- 为 `.modal-overlay` 添加 `backdrop-filter: blur(4px)`
- 为 `.modal-card` 添加内边框 `1px solid rgba(255,255,255,0.06)`
- 移除未使用的 `textInputFadeIn` 动画
- 移除 HTML 中的内联样式（L56 `style="margin-left:auto"` → CSS 类）

### 7. 不修改的内容
- `stitch-engine.js` 中 canvas 绘制的图标（需要硬编码路径）
- `main.js` 中的 grip handle SVG 和 cursor data URI（功能性）
- `floating-toolbar.js`（已正确使用 `createIcons()`）
- 布局算法和核心功能逻辑

## 验证步骤
1. `npm run dev` — 应用加载无错误
2. 检查所有图标正确渲染（无空白/缺失图标）
3. 检查字体加载正确
4. 功能测试：添加图片、撤销/重做、保存弹窗、信息弹窗
5. `npm run build` — 构建成功
